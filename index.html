Tooltips

Show button and control hints throughout Signboard.

Daily due date reminders

Signboard checks open boards each day at your selected local time and notifies you if cards are due.

Use HH:MM in 24-hour format. Hours can be 01 through 24 (example: 21:30).

Board General

Renaming updates this board folder name in its current location.

Move this board folder to another location on your computer.

Workflow

Completed lists are hidden from Planner date views unless completed cards are shown in Planner filters.

Auto-detect completed lists

Lists named Done, Completed, Complete, Closed, Finished, Resolved, or Shipped are treated as completed automatically.

Completed lists

Check any list that should count as finished work. Auto-detected lists can be unchecked here.

Import

Note: This feature is in rapid improvement. Import data from other apps into this board. Signboard copies data in and leaves your source files where they are.

Trello

Export from Trello by opening the board menu, choosing Print > Export > Share, and then exporting the board as a JSON file.

Obsidian

Choose one markdown file, several markdown files, or an entire Obsidian folder. Signboard will try to preserve lists, cards, order, dates, labels, and extra metadata.

Tasks.md

Choose a Tasks.md project folder. Signboard imports lanes, markdown body, `[tag:...]` markers, `[due:YYYY-MM-DD]` markers, and will also preserve tag colors/order when it can find the companion Tasks.md config directory.
